Installing Games

If you need further help, ask on the official Cemu Discord Server for assistance.

This page will take you through installing decrypted game dumps in Cemu. Ensure that you have your main game, the latest update, and any DLC wanted first beforehand.

If you have an encrypted game dump , you may need to decrypt it using Cdecrypt. If you used dumpling to get your dump, you won't need to do this.

WARNING

Some games, such as Breath of the Wildopen in new window, require the latest update to function correctly in Cemu.

# Installing titles

If you have a game dump, you can install it through Cemu in the "Installing dumps" tab. If you have dumped your account data and online files, follow the "Downloading from eShop" tab to download and install your game.


  1. Open the Cemu application
  2. On the top bar, click Tools -> Download Manager
  3. Select your account, then click "Connect"
  4. Right click on the title you want to install, and click "Download"

Once installed, you should be able to launch the title from the Cemu menu.

  1. Open the Cemu application
  2. On the top bar, click File -> Install game title, update or DLC
  3. If you used dumplingopen in new window to dump your game, navigate to the dumpling folder on your SD card
    • If you used an alternative dumping tool, navigate to the folder where you keep your game dumps
  4. Open the relevant folder corresponding to what you want to install
    • This is either the Games, Updates or DLC folder for dumpling
  5. Open the folder of what you'd like to install
  6. Open the meta folder and select meta.xml
  7. Cemu should now automatically install the files for you

Repeat this as many times as you need until everything you dumped is installed.

Linux users may have issues installing updates/DLC. You may have to extract the title directly to the installation directory: mlc01/usr/title. If using Lutris, please read the notesopen in new window under the script provided.

# Saves

  1. Open the Cemu application
  2. Right click on the game you'd like to transfer the save to
  3. Click Save directory
  4. If you used dumpling, open the dumpling/Saves/[game] folder from your SD card or USB device
    • If you used an alternative dumping tool, navigate to the folder where you keep your save backups
  5. Copy and paste the contents to the save directory you opened from Cemu

You should now have all the games you dumped installed onto Cemu.
